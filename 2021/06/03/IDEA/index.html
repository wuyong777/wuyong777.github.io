idea 导入 eclipse 项目-SSM

  1. 参考：B站-杰凡it

  2. 直接 open 打开项目

  3. 发现报错(红线)

  4. 设置 project structure

    1. 进入 project-project sdk 配置 jdk
    2. 进入 modules-dependencies 删掉 eclipse 多余的配置
    3. 在 modules-dependencies 添加 tomcat 依赖(右上角+号)
    4. 在 libraries 添加 jar 包依赖
    5. 在 modules 中添加 web 模块(1.左上角+号。2.注意修改路径：web 改为 WebContent)
    6. 配置 artifacts(artifacts–>web application exploded–>from modules)

  5. 将项目添加到 tomcat 中

    1. edit configurations–>左上角+号–>deployment–>右上角+号–>artifact
    2. 如果项目中文件上传用的是绝对路径的话需要配置虚拟路径(例如存放图片的目录)

  6. 修改数据库配置文件

    1. 数据库配置成自己本地的(driver,url,username,password等等)
    2. navcat 新建一个数据库(数据库和源代码一致)
    3. 导入运行 .sql 文件(建表和插入数据)

  7. 启动 tomcat

  8. 浏览器访问项目

idea 导入 eclipse 项目-Springboot

  1. 参考：B站-杰凡it
  2. 直接 open 打开项目
  3. 删除带有 eclipse 图标的 .classpath 和 .project l两个文件(不限碍眼的话可以不删)
  4. 设置 project structure(步骤跟上面导入SSM差不多，细节请参考B站-杰凡it)
  5. 导入 maven 依赖(pom.xml 右键点击 add as maven project)
  6. 配置数据库(注意：pom.xml 的 mysql jar 包版本要兼容，不兼容要修改版本并刷新)
  7. 测试运行
  8. 浏览器访问

idea 导入 eclipse 项目总结(主要步骤)

  1. 先全局配置好 idea 的 jdk , maven , git 或 svn , tomcat 等等，mysql , navcat 安装好
  2. open 打开项目
  3. 配置 project structure
  4. 配置 add/edit configurations
  5. 配置数据库
